# Getting Started

Get up and running with Repak X in minutes.

## Download

Grab the latest build from the [**Releases page**](https://github.com/XzantGaming/Repak-X/releases).

## Installation

1. Download the latest `.exe` app from the releases page
2. Run the app and follow the on-screen instructions

## First Launch

On first launch, Repak X will:

1. **Show the onboarding tour** — a guided walkthrough of the interface
2. **Prompt you to set your game path** — a warning will appear if your Marvel Rivals installation path isn't configured. Head to **Settings** and use the **Auto-Detect** button or manually set the path
3. **Set up your mod library** — you can now start installing mods

## System Requirements

- **OS:** Windows 10/11 (64-bit)
- **Marvel Rivals** installed (Steam or Epic Games)
