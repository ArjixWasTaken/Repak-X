# FAQ

Frequently asked questions about Repak X.

## General

### What is Repak X?

Repak X is a desktop application for installing, managing, and sharing Marvel Rivals mods. It supports both Legacy PAK and IO Store bundle formats.

### Is Repak X free?

Yes, Repak X is completely free and open source under the MIT license.

### What platforms does Repak X support?

Repak X currently supports Windows 10 and Windows 11 (64-bit).

### Does Repak X work with Epic Games?

Yes. Mod management works with both Steam and Epic Games installations. The only Steam-specific feature is the **Launch Game** button, which uses the Steam console.

## Mods

### What mod formats are supported?

Repak X supports `.pak` files, `.utoc`/`.ucas` IO Store bundles, `.uasset` files, folders, and common archive formats (`.zip`, `.rar`, `.7z`).

### How do I install a mod?

Simply drag and drop your mod files onto the Repak X window, or use the file picker. See the [Installing Mods](./guides/installing-mods) guide for details.

### Can I install multiple mods at once?

Yes! You can drop multiple files or folders at once and configure each mod individually before installing.

### What happens when mods conflict?

Repak X includes a global clash detection system that identifies conflicts and lets you choose which mod takes priority.

## Troubleshooting

### The app won't start

Make sure you have the latest version installed from the [Releases page](https://github.com/XzantGaming/Repak-X/releases). If the issue persists, join the [Discord](https://discord.gg/nrud2gjUJk) for support.

### My mods aren't loading in-game

Ensure your game directory is set correctly in the app settings and that mods are enabled (toggled on) in the mod list.
