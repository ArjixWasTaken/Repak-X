# Game Integration

Repak X integrates directly with Marvel Rivals for a seamless modding experience.

## Smart Crash Detection

The app monitors the game process and detects crashes, helping you identify problematic mods.

## Steam Launch

Launch Marvel Rivals directly from Repak X through Steam. This feature uses the Steam console and is only available for Steam installations.

## Mod Operations Lock

When Marvel Rivals is running, mod operations are locked to prevent file corruption. You can still browse and organize your library, but installation and removal are paused until the game closes.

## Discord Rich Presence

Show off what you're modding with Discord Rich Presence integration.
