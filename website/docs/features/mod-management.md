# Mod Management

Organize and manage your mod library with powerful tools.

## Search & Filtering

- **Search bar** — find mods by name
- **Hero filtering** — filter by Marvel Rivals character
- **Categories & Tags** — organize mods into groups
- **Folder management** — group mods by folder

## View Modes

Switch between **Card view** and **List view** for your mod library.

## Mod Details Panel

Click any mod to open the details panel, which includes:

- Mod name, tags, and metadata
- Built-in **Asset Tree browser** to inspect mod contents
- Quick actions (enable, disable, update, delete)

## Bulk Operations

Select multiple mods for batch actions like enabling, disabling, or deleting.

## Context Menus

Right-click any mod for quick actions including copy path, open in explorer, and more.
