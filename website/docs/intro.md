# Introduction

Welcome to the **Repak X** documentation — the definitive Marvel Rivals mod installer and manager.

Repak X is a next-gen Windows desktop application built with **Tauri**, **React**, **Rust**, and **C#** that makes installing, organizing, and sharing Marvel Rivals mods effortless.

## What is Repak X?

Repak X handles everything you need for Marvel Rivals modding:

- **Install mods** from PAK files, folders, or archives with drag & drop
- **Manage your mod library** with search, filtering, categories, and tags
- **Share mods** with encrypted modpack sharing
- **Protect your mods** with AES-256 encryption and obfuscation
- **Detect conflicts** between mods with the clash detection system
- **Auto-detect** character, category, and mod type
